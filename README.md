# free_stuff@jku

**Static frontend hosted on GitHub Pages + Supabase backend.**

- **Frontend**: `/docs` folder — served by GitHub Pages
- **Backend**: Supabase (PostgreSQL database + REST API)

## Quick Start

1. Read [docs/README.md](docs/README.md) for Supabase setup
2. Add your Supabase URL and key to `docs/script.js`
3. Click **Go Live** in VS Code to test locally
4. Push to GitHub — Pages auto-deploys from `/docs`

## Development

- Edit HTML files in `/docs` for pages
- Edit `docs/script.js` for Supabase API calls
- Edit `docs/styles.css` for styling
- Live Server reloads automatically on save

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript (ES6)
- **Backend**: Supabase (PostgreSQL + REST API)
- **Hosting**: GitHub Pages (static)
